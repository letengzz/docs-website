# cnpm 包管理工具

cnpm 是一个淘宝构建的 npmjs.com 的完整镜像，也称为淘宝镜像

- 网址：https://npmmirror.com/ 

cnpm 服务部署在国内 阿里云服务器上 ， 可以提高包的下载速度 官方也提供了一个全局工具包 cnpm ，操作命令与 npm 大体相同

## 安装cnpm

通过 npm 来安装 cnpm 工具：

```shell
npm install -g cnpm --registry=https://registry.npmmirror.com
```
