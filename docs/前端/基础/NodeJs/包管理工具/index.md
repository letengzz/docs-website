# 包管理工具

『包』英文单词是 package ，代表了一组特定功能的源码集合

管理『包』的应用软件，可以对「包」进行下载安装， 更新， 删除， 上传等操作

**借助包管理工具，可以快速开发项目，提升开发效率**

包管理工具是一个通用的概念，很多编程语言都有包管理工具，所以掌握好包管理工具非常重要。

**常用的包管理工具**：

- [npm 包管理工具](npm.md)
- [cnpm 包管理工具](cnpm.md)
- [yarn 包管理工具](yarn.md)
